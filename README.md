# JADAI — Landing

Landing oficial de **JADAI**, estudio de IA aplicada (Guayaquil, Ecuador).
Stack: **Next.js 16 + Tailwind v4 + React Three Fiber**, deploy en **Vercel**. Deadline: **18 jul 2026**.
